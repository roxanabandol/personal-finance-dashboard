# Personal Finance Dashboard

## Description

A modern dashboard for tracking personal expenses, built with **React**, **TypeScript**, and **TailwindCSS**, featuring **JSON Server persistence**, **Recharts graphs**, **Framer Motion animations**, and **dark mode**.  
Ideal for portfolio use, demonstrating frontend skills and state management with **Zustand**.

---

## Features

- Add, view, edit and delete expenses
- Dashboard with **statistic cards** (Total, Food, Transport)
- **Bar Chart** visualizing expenses by category
- Filter expenses by category
- Toggle between light and dark mode
- Smooth **Framer Motion** animations
- Data persistence via **JSON Server**

## Installation & Setup

1. Clone the repository:

```bash
git clone <YOUR_REPO_URL>
cd personal-finance-dashboard
```

2. Install dependencies:

```bash
npm install
```

3. Start JSON Server (fake REST API):

```bash
npm run server
```

4. Start the React app:

```bash
npm start
```
